# Nrtur Design Prototypes

This repository contains the initial prototype state for Nrtur's design system and CRM screens.

The current files are a work in progress. They are useful as a starting point for implementation, but there is still a lot of improvement waiting across layout, interaction polish, responsiveness, and production readiness.

Primary design assets and prototypes live under `nrtur-design-system/`.

If you are building from this repo, treat these files as the baseline reference rather than a finished product.
